# E-Ink Pomo
This project is an effort to learn more about niche e-ink technology. 

### Purpose
The _E-ink Pomo_ device facilitates a digital minimalist philosophy by providing functionality as an external timer for your studying or working, distraction-free environment while staying true to the theme of mindfulness with an e-ink display.
